# Trading Protocol - Real Money Rules
- **Liquidity:** Min $2B Market Cap, 1M+ Avg Daily Vol.
- **Math:** Use Mansfield RS for rotation. No "Vibe" analysis.
- **Verification:** 2+ sources for every price point.
- **Risk:** Always calculate ATR-based stops. 
- **Audit:** Save a screenshot and log for every recommendation.

# Alternative Data Rules
- **Insider Activity:** Check Form 4s (60-day window). CEO/CFO buys = +15 to Conviction Score.
- **Sentiment Scrape:** Must analyze Reddit + X + News. Compare retail hype vs. institutional price action.
- **FCS Threshold:** Minimum 75/100 score required for "Real Money" recommendations.
- **Divergence Check:** If Sentiment is High but Insiders are Selling, trigger a "Red Flag" report.
